📡 Communication Protocols

🔹 UART (Universal Asynchronous Receiver Transmitter)

Used for serial communication between microcontroller and devices like computer.

  • Simple communication
  • No clock signal required
  • Example: Arduino to PC

🔹 SPI (Serial Peripheral Interface)

High speed communication protocol.

  • Uses 4 wires (MOSI, MISO, SCK, SS)
  • Very fast data transfer
  • Used in sensors, displays

🔹 I2C (Inter-Integrated Circuit)

Two-wire communication protocol.

  • Uses SDA and SCL
  • Multiple devices can connect
  • Used in RTC, sensors

🔧 Hardware Components

  • Arduino Board
  • Sensor Module (I2C/SPI)
  • Connecting Wires
  • LCD Display
