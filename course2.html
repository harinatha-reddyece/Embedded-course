🔌 Arduino Basics

📌 What is Arduino?

Arduino is an open-source microcontroller platform used to build electronics projects.

🔧 Hardware Components

  • Arduino Board (UNO)
  • LED
  • Resistor (220Ω)
  • Breadboard
  • Jumper Wires
  • Push Button

💡 Project 1: LED Blinking

This project makes an LED blink ON and OFF.

🧰 Components:

  • Arduino UNO
  • LED
  • 220Ω Resistor
  • Jumper wires

💻 Code:

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

🔘 Project 2: Button Control LED

Press button → LED ON, Release → LED OFF.

🧰 Components:

  • Arduino UNO
  • LED
  • Push Button
  • Resistor

💻 Code:

int button = 2;
int led = 13;

void setup() {
  pinMode(button, INPUT);
  pinMode(led, OUTPUT);
}

void loop() {
  if(digitalRead(button) == HIGH) {
    digitalWrite(led, HIGH);
  } else {
    digitalWrite(led, LOW);
  }
}
