📘 What is Embedded C?

Embedded C is a programming language used to develop software for embedded systems like microcontrollers.

🔧 Components Used

  • Microcontroller (8051 / Arduino)
  • LED
  • Resistor
  • Push Button
  • Power Supply

💻 Basic Embedded C Code

#include <stdio.h>

int main() {
    printf("Hello Embedded C");
    return 0;
}

⚡ LED Blinking Code

#include <reg51.h>

void delay() {
  int i,j;
  for(i=0;i<100;i++)
    for(j=0;j<1000;j++);
}

void main() {
  while(1) {
    P1 = 0xFF;
    delay();
    P1 = 0x00;
    delay();
  }
}
